# Convertir pulgadas a centimetros

Esta funcion convierte un valor numerico en pulgadas a centimetros.

## Usage

``` r
convertir_pulgadas_a_centimetros(pulgadas)
```

## Arguments

- pulgadas:

  Valor numerico en pulgadas.

## Value

Valor numerico convertido a centimetros.

## Examples

``` r
convertir_pulgadas_a_centimetros(10)
#> [1] 25.4
```
