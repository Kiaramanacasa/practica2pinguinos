# Datos de pingüinos de ejemplo

Conjunto de datos de ejemplo incluido en el paquete
`practica2pinguinos`. Este dataset contiene información mínima para
ilustrar el uso de las funciones.

## Usage

``` r
pinguinos_demo
```

## Format

Un data frame con 2 filas y 3 columnas:

- especie:

  Especie del pingüino (carácter).

- peso_kg:

  Peso del pingüino en kilogramos (numérico).

- altura_cm:

  Altura del pingüino en centímetros (numérico).

## Source

Datos generados específicamente para este paquete.
